# MoveNet 3D Sample

![demo](demo.gif)

Sample project showing 3D pose detection in augmented reality with NatML and ARFoundation in Unity Engine.

## Requirements
- Unity 2020.3+

## Supported Platforms
- Android API level 24+
- iOS 13+
- macOS 10.15+
- Windows 10 (64-bit)
